## Installation

```sh
npm install -g @maizzle/cli
```

## Usage

Create a project:

```sh
maizzle new
```

Change current directory:

```sh
cd maizzle
```

Start a local server by running the `serve` command:

```sh
maizzle serve
```

Build for production:

```sh
maizzle build production
```
